# Tron-Pygame
A clone of the game *Tron* using Python and Pygame

## Controls

 - Player One:
   - Up: W  
   - Down: S  
   - Left: A  
   - Right: D  
   - Boost: TAB  
 - Player Two:
   - Up: Up Arrow  
   - Down: Down Arrow  
   - Left: Left Arrow  
   - Right: Right Arrow  
   - Boost: Right Shift
